# Contar Invitaciones Pendientes

Obtiene el número total de invitaciones que has enviado y están pendientes de respuesta.

## Endpoint

```
GET /api/py-linkedin/count-pending-invitations
```

## Headers

| Header | Tipo | Requerido | Descripción |
|--------|------|-----------|-------------|
| `X-API-Key` | string | Sí | Token JWT de autenticación |

## Ejemplo de Request

<CodeGroup>

```bash cURL
curl -X GET http://127.0.0.1:8000/api/py-linkedin/count-pending-invitations \\
  -H "X-API-Key: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript
const response = await fetch('/api/py-linkedin/count-pending-invitations', {
  method: 'GET',
  headers: {
    'X-API-Key': 'your-jwt-token'
  }
});

const data = await response.json();
```

```python Python
import requests

url = 'http://127.0.0.1:8000/api/py-linkedin/count-pending-invitations'
headers = {
    'X-API-Key': 'your-jwt-token'
}

response = requests.get(url, headers=headers)
data = response.json()
```

</CodeGroup>

## Response

### Respuesta Exitosa

```json
{
  "success": true,
  "response": {
    "pendingInvitationsCount": 45
  },
  "error": null
}
```

## Campos de Respuesta

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `pendingInvitationsCount` | integer | Número total de invitaciones pendientes |

## Códigos de Error

| Código HTTP | Error Code | Descripción |
|-------------|------------|-------------|
| 401 | `UNAUTHORIZED` | Token inválido o expirado |
| 500 | `SERVER_ERROR` | Error interno del servidor |

## Uso

Este endpoint es útil para:
- Verificar rápidamente cuántas invitaciones hay pendientes
- Decidir si es apropiado enviar más invitaciones
- Monitorear el uso de la cuota de invitaciones

<Callout kind="info">
  LinkedIn limita el número de invitaciones pendientes que puedes tener simultáneamente
  (típicamente entre 100-200). Este endpoint te ayuda a monitorear ese límite.
</Callout>

<Callout kind="tip">
  Este endpoint es más ligero y rápido que `/pending-invitations` cuando solo necesitas
  el conteo total sin los detalles de cada invitación.
</Callout>
