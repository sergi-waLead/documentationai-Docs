# Invitaciones Pendientes

Obtiene un listado de todas las invitaciones que has enviado y están pendientes de respuesta.

## Endpoint

```
GET /api/py-linkedin/pending-invitations
```

## Headers

| Header | Tipo | Requerido | Descripción |
|--------|------|-----------|-------------|
| `X-API-Key` | string | Sí | Token JWT de autenticación |

## Query Parameters

| Parámetro | Tipo | Default | Min | Max | Descripción |
|-----------|------|---------|-----|-----|-------------|
| `start` | integer | `0` | 0 | - | Offset de los resultados (paginación) |
| `count` | integer | `20` | 1 | 100 | Número de invitaciones a recuperar |
| `checkLimits` | boolean | `false` | - | - | Verificar límites de uso |

## Campos de Respuesta

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `total` | integer | Número total de invitaciones pendientes |
| `invitations` | array | Lista de invitaciones pendientes |

### Objeto Invitation

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `invitationUrn` | string | URN de la invitación (sin prefijo) |
| `entityUrn` | string \| null | URN del perfil invitado (sin prefijo) |
| `publicId` | string \| null | ID público del perfil |
| `name` | string \| null | Nombre completo del perfil |
| `headline` | string \| null | Titular profesional |
| `pictureUrl` | string \| null | URL de la foto de perfil |
| `sentTime` | integer \| null | Timestamp de envío (milisegundos) |

## Paginación

Para paginar los resultados, usa los parámetros `start` y `count`:

```javascript
// Primera página (primeras 20 invitaciones)
GET /pending-invitations?start=0&count=20

// Segunda página (siguientes 20 invitaciones)
GET /pending-invitations?start=20&count=20

// Tercera página
GET /pending-invitations?start=40&count=20
```

## Códigos de Error

| Código HTTP | Error Code | Descripción |
|-------------|------------|-------------|
| 401 | `UNAUTHORIZED` | Token inválido o expirado |
| 422 | `VALIDATION_ERROR` | Parámetros fuera de rango |
| 500 | `SERVER_ERROR` | Error interno del servidor |

## Uso

Este endpoint es útil para:
- Monitorear invitaciones enviadas
- Identificar invitaciones antiguas para retirar
- Gestionar tu cuota de invitaciones pendientes
- Analizar tasas de aceptación

<Info>
  LinkedIn limita el número de invitaciones pendientes que puedes tener.
  Retira invitaciones antiguas para liberar cuota.
</Info>

<Tip>
  Combina este endpoint con `/check-invitation` para obtener el estado
  actualizado de cada invitación.
</Tip>
