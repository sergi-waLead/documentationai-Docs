# Buscar en Sales Navigator

Busca leads utilizando LinkedIn Sales Navigator, ofreciendo capacidades de búsqueda avanzadas.

## Endpoint

```
POST /api/py-linkedin/search-sales-navigator
```

## Headers

| Header | Tipo | Requerido | Descripción |
|--------|------|-----------|-------------|
| `X-API-Key` | string | Sí | Token JWT de autenticación |
| `Content-Type` | string | Sí | `application/json` |

## Query Parameters

| Parámetro | Tipo | Default | Descripción |
|-----------|------|---------|-------------|
| `checkLimits` | boolean | `false` | Si es `true`, verifica y consume límites de uso |

## Body Parameters

| Campo | Tipo | Default | Descripción |
|-------|------|---------|-------------|
| `query` | string | - | Cadena de búsqueda para Sales Navigator |
| `start` | integer | `0` | Offset para paginación |
| `count` | integer | `0` | Número de resultados a obtener |

<Callout kind="note">
  La estructura de la `query` depende de los filtros que quieras aplicar en Sales Navigator.
  Consulta la documentación de LinkedIn Sales Navigator para los formatos específicos.
</Callout>

```bash cURL
  -H "Content-Type: application/json" \\
  -d '{

    "start": 0,

```



## Códigos de Error

| Código HTTP | Error Code | Descripción |
|-------------|------------|-------------|
| 400 | `BAD_REQUEST` | Query de búsqueda inválida |
| 401 | `UNAUTHORIZED` | Token inválido o expirado |
| 403 | `FORBIDDEN` | No tienes acceso a Sales Navigator |
| 422 | `VALIDATION_ERROR` | Error en la validación de parámetros |
| 429 | `RATE_LIMIT` | Límite de búsquedas alcanzado |
| 500 | `SERVER_ERROR` | Error interno del servidor |

## Paginación

Para paginar resultados, incrementa el valor de `start`:

```javascript
// Primera página (25 resultados)
{ query: 'engineer', start: 0, count: 25 }

// Segunda página
{ query: 'engineer', start: 25, count: 25 }

// Tercera página
{ query: 'engineer', start: 50, count: 25 }
```

## Requisitos

<Callout kind="alert">
  **Se requiere una cuenta de LinkedIn Sales Navigator activa** para usar este endpoint.
  Las cuentas regulares de LinkedIn no tienen acceso a esta funcionalidad.
</Callout>

## Ventajas de Sales Navigator

- **Filtros avanzados**: Más opciones de filtrado que la búsqueda estándar
- **Leads guardados**: Capacidad para guardar y gestionar leads
- **InMail credits**: Enviar mensajes a perfiles fuera de tu red
- **Insights**: Información detallada sobre empresas y leads
- **Alertas**: Notificaciones sobre cambios en tus leads

## Casos de Uso

- Prospección de ventas avanzada
- Identificación de tomadores de decisiones
- Búsqueda de leads calificados por industria
- Investigación de mercado B2B
- Outreach de alto valor

<Callout kind="tip">
  Sales Navigator ofrece capacidades de búsqueda significativamente más potentes
  que la búsqueda estándar, incluyendo filtros de intención de compra y technology stack.
</Callout>

<Callout kind="info">
  Los resultados de Sales Navigator suelen ser más precisos y contener más
  información que las búsquedas estándar.
</Callout>
