# GET /identity/profiles/{publicId}/profileView

Obtiene la vista completa de un perfil de LinkedIn y registra una vista del perfil.

## Endpoint

```
GET https://www.linkedin.com/voyager/api/identity/profiles/{publicId}/profileView
```

## Path Parameters

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `publicId` | string | Public identifier del perfil (ej: john-doe) |

## Ejemplo

```bash
curl -X GET "https://www.linkedin.com/voyager/api/identity/profiles/john-doe/profileView" \\
  -H "Cookie: li_at=YOUR_LI_AT; JSESSIONID=ajax:YOUR_JSESSIONID" \\
  -H "csrf-token: YOUR_JSESSIONID"
```

<Callout kind="alert">
  Este endpoint registra una vista del perfil. Úsalo con moderación para evitar alertar al usuario y exceder rate limits de LinkedIn.
</Callout>
