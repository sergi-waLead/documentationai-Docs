# POST /voyagerMessagingGraphQL/graphql

Obtiene la lista de conversaciones de mensajes usando GraphQL.

## Endpoint

```
POST https://www.linkedin.com/voyager/api/voyagerMessagingGraphQL/graphql
```

## Query ID

```
voyagerMessagingDashConversationsByCriteria.123abc
```

## Variables

```json
{
  "mailboxUrn": "urn:li:fsd_profile:ACoAAA...",
  "count": 20
}
```
