# PUT /messaging/conversations/{conversationUrn}

Actualiza el estado de una conversación (principalmente para marcarla como leída).

## Endpoint

```
PUT https://www.linkedin.com/voyager/api/messaging/conversations/{conversationUrnId}
```

## Request Body

```json
{
  "patch": {
    "$set": {
      "read": true
    }
  }
}
```
