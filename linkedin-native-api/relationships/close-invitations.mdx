# POST /relationships/invitations?action=closeInvitations

Cancela/elimina una o más invitaciones de conexión pendientes que hayas enviado.

## Endpoint

```
POST https://www.linkedin.com/voyager/api/relationships/invitations?action=closeInvitations
```

## Headers

```
Cookie: li_at={token}; JSESSIONID=ajax:{session}
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36
X-Li-User-Agent: LIAuthLibrary:0.0.3 com.linkedin.android:4.1.881
csrf-token: {session}
Content-Type: application/json
```

## Request Body

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| `invitations` | array | Sí | Array de URNs de invitaciones a cancelar |

```json
{
  "invitations": [
    "urn:li:fsd_invitation:12345678"
  ]
}
```

## Respuesta Exitosa

```json
{
  "data": {
    "value": {}
  }
}
```

HTTP Status: **200 OK**

<Callout kind="note">
  Una respuesta vacía con status 200 indica que la invitación fue cancelada exitosamente.
</Callout>

## Cancelar Múltiples Invitaciones

Puedes cancelar varias invitaciones en un solo request:

```json
{
  "invitations": [
    "urn:li:fsd_invitation:12345678",
    "urn:li:fsd_invitation:87654321",
    "urn:li:fsd_invitation:11223344"
  ]
}
```

<Callout kind="tip">
  LinkedIn recomienda no cancelar más de 10 invitaciones por request para evitar rate limiting.
</Callout>

## Casos de Uso

- Cancelar invitaciones antiguas (>7 días sin respuesta)
- Limpiar invitaciones antes de enviar nuevas
- Retirar invitaciones a perfiles incorrectos
- Gestionar cuota de invitaciones pendientes

## Rate Limits

- **Frecuencia recomendada**: 1 request cada 5-10 segundos
- **Max por hora**: ~20 requests
- **Max invitaciones por día**: ~50-100 cancelaciones

<Callout kind="alert">
  Cancelar demasiadas invitaciones seguidas puede alertar a LinkedIn. Usa delays apropiados.
</Callout>

## Errores Comunes

| Código | Descripción | Solución |
|--------|-------------|----------|
| 400 | Invalid invitation URN | Verificar formato del URN |
| 401 | No autorizado | Verificar cookies |
| 404 | Invitation not found | La invitación ya fue aceptada o cancelada |
| 429 | Rate limit excedido | Reducir frecuencia |

### Error: Invitation Already Accepted

Si intentas cancelar una invitación que ya fue aceptada, recibirás un error 404:

```json
{
  "status": 404,
  "message": "Resource not found"
}
```

## Obtener URN de Invitación

Para cancelar una invitación, primero necesitas su URN. Obtenerlo usando [Sent Invitations](./sent-invitations):

```python
# 1. Obtener lista de invitaciones pendientes
response = requests.get(
    'https://www.linkedin.com/voyager/api/relationships/sentInvitationViewsV2',
    headers=headers,
    params={'count': 20, 'start': 0, 'invitationType': 'CONNECTION', 'q': 'invitationType'}
)

invitations = response.json()['elements']

# 2. Cancelar invitaciones antiguas (>7 días)
import time
seven_days_ago = (time.time() - 7*24*60*60) * 1000

old_invitation_urns = [
    inv['entityUrn']
    for inv in invitations
    if inv['sentTime'] < seven_days_ago
]

# 3. Cancelar
cancel_response = requests.post(
    'https://www.linkedin.com/voyager/api/relationships/invitations?action=closeInvitations',
    headers=headers,
    json={'invitations': old_invitation_urns}
)
```
