# GET /relationships/sentInvitationViewsV2

Obtiene una lista paginada de todas las invitaciones de conexión enviadas que están pendientes de aceptación, con detalles completos de cada perfil.

## Endpoint

```
GET https://www.linkedin.com/voyager/api/relationships/sentInvitationViewsV2
```

## Query Parameters

| Parámetro | Tipo | Requerido | Descripción |
|-----------|------|-----------|-------------|
| `count` | integer | No | Número de invitaciones por página (default: 20) |
| `start` | integer | No | Índice inicial para paginación (default: 0) |
| `invitationType` | string | Sí | Tipo de invitación. Usar `CONNECTION` |
| `q` | string | Sí | Query type. Usar `invitationType` |

## Headers

```
Cookie: li_at={token}; JSESSIONID=ajax:{session}
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36
X-Li-User-Agent: LIAuthLibrary:0.0.3 com.linkedin.android:4.1.881
csrf-token: {session}
```

```python Python
import requests

li_at = "YOUR_LI_AT"
jsessionid = "YOUR_JSESSIONID"

headers = {
    'Cookie': f'li_at={li_at}; JSESSIONID=ajax:{jsessionid}',
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'X-Li-User-Agent': 'LIAuthLibrary:0.0.3 com.linkedin.android:4.1.881',
    'csrf-token': jsessionid
}

params = {
    'count': 20,
    'start': 0,
    'invitationType': 'CONNECTION',
    'q': 'invitationType'
}

response = requests.get(
    'https://www.linkedin.com/voyager/api/relationships/sentInvitationViewsV2',
    headers=headers,
    params=params
)

data = response.json()
for invitation in data['elements']:
    print(f"Invitation to: {invitation['toMember']['firstName']} {invitation['toMember']['lastName']}")
```

### Campos de Respuesta

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `entityUrn` | string | URN único de la invitación |
| `invitationType` | string | Tipo de invitación (CONNECTION) |
| `invitationState` | string | Estado: `PENDING`, `ACCEPTED`, `WITHDRAWN` |
| `sentTime` | integer | Timestamp en milisegundos del envío |
| `toMember` | object | Información del perfil invitado |
| `message` | string | Mensaje personalizado enviado (si existe) |
| `paging.total` | integer | Número total de invitaciones pendientes |

## Paginación

Para obtener la siguiente página:

```python
# Segunda página (invitaciones 20-39)
params['start'] = 20

# Tercera página (invitaciones 40-59)
params['start'] = 40
```

<Callout kind="tip">
  Usa `paging.total` para calcular el número total de páginas: `total_pages = ceil(total / count)`
</Callout>

## Casos de Uso

- Ver lista completa de invitaciones pendientes
- Obtener información de perfiles que no han aceptado
- Identificar invitaciones antiguas para cancelar
- Analizar tasa de de aceptación por mensaje o perfil

## Rate Limits

- **Frecuencia recomendada**: 1 request cada 2-3 minutos
- **Max requests por hora**: ~20
- **Throttling**: Evitar consultar todas las páginas de una vez

<Callout kind="alert">
  No consultes este endpoint muy frecuentemente. LinkedIn monitorea este tipo de actividad.
</Callout>

## Filtrado

Este endpoint no soporta filtrado avanzado. Para buscar invitaciones específicas, tendrás que implementar el filtrado client-side:

```python
# Buscar invitaciones enviadas hace más de 7 días
import time
seven_days_ago = (time.time() - 7*24*60*60) * 1000

old_invitations = [
    inv for inv in data['elements']
    if inv['sentTime'] < seven_days_ago
]
```

## Errores Comunes

| Código | Descripción | Solución |
|--------|-------------|----------|
| 401 | No autorizado | Verificar cookies |
| 422 | Start index out of range | Reducir valor de `start` |
| 429 | Rate limit excedido | Esperar varios minutos |
